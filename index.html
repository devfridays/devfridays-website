Sobre

O DevFridays é um meetup formado em 2017 por um grupo de amigos que queriam fomentar a comunidade local de desenvolvimento e compartilhar conhecimento e experiencias sobre desenvolvimento em Campo Grande - MS.

FB Group

Meetup

Também conhecido como DevFridays, desde a primeira edição em 2017 o encontro ocorre todas as sextas-feiras. Os encontros são totalmente gratuitos e sempre contam com o apoio de alguma(s) empresa(s), tanto para o local quanto para pizzas e aquele chopp! Participe como quiser dos nossos meetups, seja como participante, talker ou empresa apoiadora. Nosso foco é fortalecer a comunidade de desenvolvimento de Campo Grande - MS.

github

Como forma de retribuir a comunidade tudo aquilo que usamos hoje, temos nosso github ainda engatinhando nos projetos open-source. Precisamos de um time grande e forte, para assim criar novos projetos e dar manutenção nos já existentes. Se você deseja contribuir com nossos projetos, trazer um projeto novo para nosso github ou somente fazer parte do time, entre em contato com alguns dos devs que lá estão ou pelo e-mail contato@devfridays.github.io.
